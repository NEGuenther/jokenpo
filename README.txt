# Jokenpo (Pedra, Papel e Tesoura)

Este projeto é um jogo simples de Jokenpo (Pedra, Papel e Tesoura) desenvolvido em HTML, CSS e JavaScript, onde você joga contra a máquina.

## Funcionalidades

- Interface amigável e responsiva.
- Três botões para escolher: Pedra 🥊, Papel 🖐️ ou Tesoura ✌️.
- A máquina faz uma escolha aleatória a cada rodada.
- O resultado da rodada é exibido na tela.
- Placar atualizado automaticamente para o jogador e para a máquina.

## Como jogar

1. Abra o arquivo `index.html` em seu navegador.
2. Clique em um dos botões para escolher Pedra, Papel ou Tesoura.
3. Veja o resultado da rodada e acompanhe o placar.

## Estrutura do projeto

- `index.html` — Estrutura da interface do jogo.
- `style.css` — Estilos para deixar o jogo mais bonito e agradável.
- `script.js` — Lógica do jogo (escolha do jogador, escolha da máquina, verificação do vencedor e atualização do placar).

## Objetivo

O objetivo é se divertir jogando Jokenpo contra a máquina e tentar vencer o maior número de rodadas possível!
